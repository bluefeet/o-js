o.js

Javascript utility belt for objects and prototypes.

Synopsis

var obj = {
    getName: o.reader('name'),
    setName: o.writer('name'),
    hasName: o.predicate('name'),
    clearName: o.clearer('name'),
    age: o.accessor('_age')
};

obj.setName('Tiberius');
if (obj.hasName()) console.log( obj.getName() );
obj.clearName();

obj.age( 23 );
console.log( obj.age() );

Description

This library provides utilities for building rich objects and prototypes. The interface is meant to be simple and flexible. No framework is provided, instead a toolbelt full of utilities is provided to make this happen. This way you can use this library the way you see fit.

Properties

Properties have definitions which describe how the property should behave. A definition is just an object containing flags and value. Not all property functions handle all definitions.

definition.default = value || function

Specifies the default value for the property when a reader is called. If no value has been set for the property then the default will be used to writer the value before it is returned. If the default is a function then the function will called, with the context set to the property's context, and the return value will be used to set the property. This way property values may be constructed only on an as-needed basis.

definition.reader = o.reader(...)

o.reader( key, [definition] )

var def = {
    // Specify a default value for this property as a plain value,
    default: 42,
    // or as a function:
    default: function () { return 42; },

    // Specify a writer to use when a default is used.
    Optional, a writer will be 
    writer: o.writer('name'),
};
Creates a reader function that will return the value of the key.

o.writer( key, [definition] )

Creates a function that will write the passed in value to the key.

o.accessor( key, [definition] )

This creates a function that is a writer if a value is passed, and a reader if a value is not passed. Any definition flags that reader and writer supports may be specified.

o.predicate( key )

Creates a a function that will return true if the property has been set.

o.clearer( key )

Creates a function that will clear the key.
